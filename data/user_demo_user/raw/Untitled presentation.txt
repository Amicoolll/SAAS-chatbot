Queues
 Queue is a data structure work on the basis of FIFO.
Example of Queues = a queue in from of ticker window, Uber sharing ride etc.
Complexity of Queues:=
Look up = O(n)
Enqueue = O(1)
Dequeue = O(1)
Peek = O(1)

 

 
Q. Why we should not array for implementing the queues?
Ans. Because every time we have to delete the first index and in array we have to shift all the index if deleted it will be O(n) complexity, With linked list we just have to delete the head it will be O(1) operation.
