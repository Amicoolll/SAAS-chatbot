As I am on leave on 18 Feb and it is important to provide the update of QEPrize, I wanted to share the current technical status


Achieved: 
• We are able to place objects and add basic shapes such as circles and squares like QEPrizze does.
• Objects can be layered and positioned above one another successfully.
• Model interaction is working as expected.
• 360-degree viewing is achievable.
• All the above features are currently stable within a 2D implementation.
Experimentation: 
• When attempting similar workflows in 3D, stability and reliability are not consistent with the current setup.
• We evaluated multiple libraries; however, many of them support React Native versions below 0.74, and strict version alignment is required with the latest toolchain.
• Due to these compatibility constraints, the 3D implementation is facing technical limitations despite continuous experimentation with alternative libraries.
Evaluation:
• Based on the current evaluation, Unity appears to be a more stable and scalable approach for achieving production-level 3D capabilities.
• I have also posted twice on the Appinventiv group to check if anyone has hands-on Unity development experience; however, so far we have only received contractor contacts and no internal expertise has been identified.
We will continue exploring feasible options and will keep you updated on further progress.
