✅ Backend (Business Logic, GPT Handling, Uploads)
Component
	Tech
	Why
	Framework
	FastAPI (Python)
	Fast, async, easy Swagger docs, works well with GPT APIs
	LLM Integration
	OpenAI GPT-3.5 (via openai Python SDK)
	Reliable, powerful for user story generation
	Document Parsing
	python-docx, PyMuPDF or pdfplumber
	Handle both DOCX and PDF with accurate text extraction
	Auth (if needed)
	JWT (via fastapi-jwt-auth)
	Optional for login/user role-based SaaS
	Rate Limiting
	slowapi or Redis-based limiter
	Prevent abuse of GPT API quota
	Background Processing
	Celery + Redis
	For longer documents (async processing)
