1. Introduction
Expin is a mobile application which allows Instagram influencers to book experiences with their selected Merchants
   1. Purpose
To develop a mobile application (for iOS and Android platform) which allow Instagram influencers to browse all the available experiences on their desired category/ merchants and book it. Once booking has been completed, Influencer posts content on their instagram channel via stories or posts, of which we will need to capture the content and the insights and save the content/insights on the web merchant portal, the influencer is also then required to submit their review against experience and vice versa.
Merchants can create experience and set up restrictions about which influencers can see and access which experiences based on different properties - like number of followers -  of the Influencer (hereinafter referred to as visible and invisible restrictions). Merchants can use the mobile app to view the influencer requests for bookings. Merchant can accept or reject the influencer booking like Tinder-card style. More over via web based dashboard, Merchants can get real time insights about the Influencers’ Instagram account and how their Posts/Stories posted about the Experiences perform.

   2. Definitions
Term
	Definitions
	Package 
	Refers the subscription plan which comes with a specific number of influencer bookings allocated per month
	Slot
	Refers the available date & time for influencer to book the experience. 
A slot is the availability within an experience which can be booked by an influencer
	Credit
	Refers the available count of influencer booking
A credit is used to open a slot within an experience, and credit is ONLY removed when a PIN is entered successfully
	Waiting list
	Refers the influencer booking need approval from Merchant done on the app or the web
	Available
	Refer the booking is available to book it immediately without merchant approval
	Check-In
	Refers the influencer’s physical presence the experience’s location at the time of the booked experience 
	PIN
	A 4-digit number given to the merchant, this number is entered on the influencer’s phone when they click on Check-in. The PIN number basically verifies that the influencer was at the venue, and that the merchant did service them
	Story
	A short vertical image/video which describes an experience.  For each Experience multiple Stories can be added by the Merchant.
Inside the Influencer section of the Mobile App Influencers can browse these Stories grouped by Experiences in a way Instagram users can browse Stories of other users they are following. Each Stories link to the related Experience. They are like a preview of the Experience they are referring to.
	Notice Period
	the minimum period of time needed between (today) and the slot date
	Action Required
	Within the notification tab, this is where we keep the tasks that the user need to do, there are now two primary tasks for influencers: 
1. Write a review on merchant post booking, 
2. Choose a timeslot for an approved waiting list request.


There are now two primary tasks for merchants: 
1. Write a review on influencer post booking 
2. Approve/reject influencers on waiting list
	Lock
	Notify the influencer that they can’t book the experience due various use cases which was described in the following sections 
	Openings
	Number of concurrent upcoming bookings an influencer can book simultaneously
	Influencer
	These are the person who are having their account in Instagram social media and promote content about various merchant and/or brands to their followers 
	Content
	is what the influencers posts online to their followers (the public) via instagram post or instagram story
	Reviews
	These are comments posted by Influencer about their experience or by Merchant about the Influencer. These reviews are only visible to our end users (merchants/influencers) and not the public.
	Insights
	refers the various stats captured from instagram for the content about the experience influencer has posted in their instagram profile 
	Experience Type


	Refers the Experience required Merchant approval or auto booking is allowed. Also it refers the Experience required the influencer check-in or content post is enough


	Experience
	Refers the free try provided by hotest brands/ Merchant
	Merchant
	These are the vendor who are wish to promote their brand information to the social influencers to achieve their brands ongoing success 
	Booking
	Refers the confirmation of the time slot is booked by the Influencer

	Influencer user must verify their profile in order to book the experience via Expin application
	

Solution Specification
1. Influencer user must verify the following to complete their profile in the Expin application to get access to experience booking feature 
   1. Mobile Number
   2. Email Address
2. Influencer user must connect their business account to complete the profile in Expin application 
3. The profile verification process is split into the following 5 steps 
   1. Step 1: Phone Number Verification 
   2. Step 2: Email Verification 
   3. Step 3: Business Account Connect
Step 1: Phone Number Verification 
1. First, user needs to verify their mobile number in order to get their profile verified status. 
2. Once user clicked on ‘Verify’ button near to Mobile No filed in their profile screen, triggers an SMS 4-digit verification code via twilio to the given phone number and user will be redirected to the screen where influencer can input the verification code received through SMS 
3. User is provide an ability to request the verification SMS again by simply tapping the ‘Resend’ option or to edit the mobile number they input before re-sending, if they edit the number, it needs to be updated in the database.
4. Verify Phone Number: Clicking this action button, after entered the verification code will validate the entered code is correct or not 
   1. If user entered code is correct,
      1. User shall be redirected to the profile page with ‘Verified’ icon near to the mobile no field instead of ‘Verify’ button 
   2. If user entered code is incorrect
      1. User get an popup notification saying that entered wrong verification code 
      2. Clicking ‘OK’ button on the popup, 
         1. Closes the popup notification window 
         2. Shows verification code screen and prompt the user to entered the pin again 
Step 2: Email Verification 
1. Influencer’s email address must be unique in the Expin application. 
2. Once user clicked on ‘Verify’ button near to Email filed in their profile screen,  User will get an popup window saying verification email sent to that address 
3. Clicking the link in the email received, will automatically update the ‘Verified’ status icon near to the Email field instead of ‘Verify’ button
Step 3: Facebook Business Account Connect 
User will be redirected to this screen, when clicking ‘Connect’ option from their profile screen. In this screen user can see the following 
1. Option to connect with Facebook Business account: There are the following 3 cases are possible when clicking on this button 
   1. Case 1: User is previously connected/ Logged into their Facebook 
      1. User is redirected to the Facebook confirm login window/screen with ‘Continue’ button. Tapping this button shall redirect the user to the profile with ‘Connected’ status icon near to Connect your account field instead of ‘Connect’ button 
   2. Case 2: User hasn’t logged into their Facebook 
      1. User is redirected to Facebook Login page 
      2. User will be asked to enter Facebook login credentials to connect 
      3. Click on “login” button should check the email address & password supplied by user is correct & registered with Facebook.
         1. If valid, then do the following
            1. Closes the popup Facebook window 
            2. User shall be redirected to profile with ‘Connected’ status icon near to Connect your account field instead of ‘Connect’ button 
         2. If not valid, display an error message and prompt user to provide the valid email address & password.
   3. Case 3: User has connected their Facebook account which is not a Business account 
      1. User will get a popup notification saying that they need to upgrade their account to business account along with how to video in it 
      2. User’s account is still remains  in ‘Unlocked’ status

	This section shows all the stories available in the Expin application.  


Stories are short vertical images and/or videos which describe the experience. User shall able to play the story video and see the complete details about that experience 
	

Solution Specification
1. All the stories added in the application shall displayed in this section 
2. Visible/ Invisible Restrictions also applied for stories view too. i.e Stories view only show the experiences that the logged in influencer is eligible to view will be displayed 
3. Stories navigation works as exactly like Instagram stories 
   1. Story images last for 4 seconds and move on to the next 
   2. Story videos are last up to 15 seconds and move on to the next 
   3. Each story can have multiple images/videos within it

      * Manage_pages
